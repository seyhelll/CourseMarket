# 📚 CourseMarket

CourseMarket — это образовательная платформа в формате мобильного приложения для iOS, объединяющая студентов, желающих получить новые знания, и авторов, готовых делиться своей экспертизой. Пользователи могут покупать, продавать и проходить онлайн-курсы в удобном интерфейсе с полной синхронизацией с удалённым сервером.

---

## 🔧 Стек технологий

### 📱 Клиент (iOS)

- SwiftUI — построение UI
- SwiftData — локальное хранилище (кэширование курсов и прогресса)
- NavigationStack — маршрутизация
- Combine + URLSession — работа с REST API
- AsyncImage, кастомные ViewModel'ы, Query, Model
- AVKit / AVPlayer — воспроизведение видеоуроков

### 🌐 Сервер (backend)

- Go (Golang) — серверная логика
- PostgreSQL — база данных
- Gin — HTTP-фреймворк
- JWT — аутентификация и авторизация
- GORM — ORM для Go

## ✨ Возможности

### 🔐 Аутентификация

- Регистрация / Вход через email и пароль
- Выбор роли: студент или автор курсов
- Поддержка "Запомнить меня"
- JWT авторизация с сохранением токена
- Проверка email на уникальность (сервером)
- Локальное кэширование профиля

### 👤 Профиль пользователя

- Просмотр и редактирование email / пароля
- Адаптивный режим интерфейса: просмотр / редактирование
- Безопасный выход (удаление токена)
- Отображение роли (student / author)

### 📚 Курсы (для студентов)

- Получение списка курсов с сервера
- Фильтрация курсов по категориям (программирование, дизайн, маркетинг и др.)
- Поиск по названию и автору
- Просмотр карточки курса: описание, программа, цена, отзывы
- Просмотр видео-превью перед покупкой
- Покупка курса (интеграция с платежным шлюзом)
- Просмотр видеоуроков после покупки
- Отслеживание прогресса обучения (шкала завершения модулей)

### ✏️ Управление курсами (для авторов)

- Добавление нового курса (название, описание, цена, категория, обложка, видео-превью)
- Загрузка видеоуроков и материалов
- Редактирование и обновление курсов
- Архивирование устаревших курсов
- Просмотр статистики продаж и отзывов

### 🛒 Корзина и оплата

- Локальное и серверное управление корзиной
- Асинхронное отображение содержимого
- Кнопка "Оплатить" с индикатором прогресса
- Интеграция с платежным шлюзом (банковские карты)
- Синхронизация корзины при повторном входе

### ⭐ Отзывы и рейтинги

- Оценка курса по шкале 1-5 звезд
- Текстовые отзывы после прохождения курса
- Отображение средней оценки в карточке курса

## 📡 REST API (сервер)

| Метод | Endpoint | Описание |
| --- | --- | --- |
| POST | /register | Регистрация пользователя |
| POST | /login | Авторизация, получение JWT |
| GET | /courses | Получение всех курсов |
| GET | /courses/:id | Получение карточки курса |
| POST | /courses | Добавление нового курса (автор) |
| PUT | /courses/:id | Обновление курса |
| DELETE | /courses/:id | Удаление / архив курса |
| POST | /courses/:id/enroll | Запись на курс (оплата) |
| GET | /my-courses | Получение купленных курсов |
| GET | /progress/:courseId | Получение прогресса по курсу |
| PUT | /progress/:courseId | Обновление прогресса |
| GET | /cart | Получение содержимого корзины |
| POST | /cart | Добавление курса в корзину |
| DELETE | /cart/:id | Удаление курса из корзины |
| GET | /profile | Получение профиля текущего юзера |
| PUT | /profile | Редактирование email / пароля |
| POST | /courses/:id/review | Добавление отзыва к курсу |

## 📦 Модели данных (iOS)

### 👤 User

struct User: Codable {
    var email: String
    var password: String
    var role: String // "student" or "author"
}

### 📖 Course

struct Course: Codable, Identifiable {
    var id: UUID
    var title: String
    var description: String
    var price: Double
    var imageUrl: String
    var authorName: String
    var authorId: UUID
    var category: String
    var trailerUrl: String
    var rating: Double
    var lessonsCount: Int
}

### 🎬 Lesson

struct Lesson: Codable, Identifiable {
    var id: UUID
    var courseId: UUID
    var title: String
    var videoUrl: String
    var duration: Int // seconds
    var order: Int
}

### 🛒 CartItem

struct CartItem: Codable, Identifiable {
    var id: UUID
    var userEmail: String
    var courseId: UUID
}

### 📊 UserProgress

struct UserProgress: Codable {
    var userEmail: String
    var courseId: UUID
    var completedLessons: [UUID]
    var percentComplete: Double
}

### 💬 Review

struct Review: Codable, Identifiable {
    var id: UUID
    var courseId: UUID
    var userEmail: String
    var rating: Int // 1-5
    var comment: String
    var createdAt: Date
}

## 🧠 Хранилище данных

- SwiftData используется для кэширования пользователя, курсов и прогресса в офлайн-режиме
- Сервер является главным источником данных
- Все действия (авторизация, покупка курса, обновление профиля, прогресс) отправляются на сервер

## 🚀 Как запустить

### 📱 iOS клиент

1. Установите Xcode 15+
2. Откройте CourseMarket.xcodeproj
3. Запустите симулятор или устройство
4. Используйте демо-аккаунт или зарегистрируйтесь

### 🌐 Сервер (Go)

1. Установите Go и PostgreSQL
2. Клонируйте репозиторий backend
3. Настройте .env с переменными БД и JWT
4. Выполните:

go run main.go
